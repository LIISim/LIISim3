CSV automatic file import

Scans the selected directory (and subdirectories if choosen) for *.csv files containing multiple signals.
The first line of the table should represent the time.

Usage:
- Select the directory which should be scanned via the 'Choose Directory' button.
- Choose the other options according to your file composition.
- Press 'Scan Files' to get a list of files matching the file name scheme.
- Check the files which should be imported and press 'Import'.

Caution: Requirements on filenames

The filenames should have the following structure:

Runame.SignalType.Tag3.ChannelID.csv

Runname: Name of the measurement run
SignalType: Absolute or Signal (raw signal data?)
Tag3: Ignored
ChannelID: Ch1 or Ch2 or Ch3 etc...

Example:
Run01.Absolute.Intensity.Ch1.csv
Run01.Signal.Volts.Ch1.csv
